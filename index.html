How to Play

🧪
Goal: Sort all liquids so each tube contains only one color.
👆
How: Tap a tube to select it, then tap another to pour. You can only pour onto a matching color or an empty tube.
🔢
Rules: A tube can only hold 4 layers. You can't pour into a full tube.
💡
Hints: Each level gives you 3 hints showing a valid move.
↩
Undo: Undo your last move any time.
⭐
Rating: Finish in fewer moves for a higher star rating!
